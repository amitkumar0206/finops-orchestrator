# IaC Workbench Demo Files

Use these files in the IaC Workbench page (`/iac`) to demo upload + analysis + chat + final-file generation.

## Files

- `terraform_unoptimized.tf`: Terraform sample with right-sizing, Multi-AZ, gp2, and lifecycle opportunities.
- `cloudformation_unoptimized.yaml`: CloudFormation sample with similar cost optimization opportunities.

## Demo Flow

1. Open the app and navigate to **IaC Workbench**.
2. Upload one sample file.
3. Review analysis, pros/cons, and estimated monthly savings table.
4. Ask follow-up chat questions such as "What changes are safe for production?".
5. Click **Generate Final File** and download original vs improved versions.
